conda常用命令

Posted on 2020-10-13

适用于任何语言的软件包，依赖项和环境管理–Python，R，Ruby，Lua，Scala，Java，JavaScript，C / C ++，FORTRAN。Conda作为软件包管理器可以帮助您查找和安装软件包。如果您需要一个需要不同版本的Python的软件包，则无需切换到其他环境管理器，因为conda也是环境管理器。仅需几个命令，您就可以设置一个完全独立的环境来运行该不同版本的Python，同时继续在正常环境中运行您通常的Python版本。

conda常用通用命令

conda clean
  1. 删除从不使用的包
    • conda clean --packages [-p]
  2. 删除tar包
    • conda clean --tarballs [-t]
  3. 删除所有删除索引缓存、锁定文件、未使用过的包和tar包
    • conda clean --all [-a]
  4. 删除缓存索引
    • conda clean --index-cache [-i]
conda create
  1. 创建新环境并指定安装xx环境
    • conda create --name envname [-n] python=3.5
  2. 创建并克隆某个环境
    • conda create --name new_env_name --clone old_env_name
conda info
  1. 列出所有已知的conda环境
    • conda info --envs [-e]
conda install
  1. 安装包
    • conda install packages_name / packages_name=3.5
  2. 在指定环境中安装包
    • conda install --name envname package_name
conda list
  1. 查看 conda安装了哪些包
    • conda list
  2. 查看 conda 当前存在哪些虚拟环境
    • conda env list 或 conda info -e
conda remove
  1. 删除创建的环境和所有软件包
    • conda remove -n envname --all
conda update
  1. 基本升级
    • conda update conda 或 conda update -n base conda
  2. 大的升级
    • conda update anaconda
  3. 更新xxx文件包
    • conda update xxx
